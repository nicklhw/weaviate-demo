# Weaviate Demo

# Quick Start

1. Install `uv`.

```
curl -LsSf https://astral.sh/uv/install.sh | sh
```

2. Create .env file.

```
WEAVIATE_URL=your-cluster-url
WEAVIATE_API_KEY=your-weaviate-api-key
OPENAI_API_KEY=your-openai-api-key
```

3. Run the demo.

```
uv run main.py
```
